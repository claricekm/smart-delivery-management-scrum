# Smart Delivery Management System

## Project Description
This academic project was developed within the scope of the practical assignment **"Desenvolvimento de um Sistema de Informação para Gestão Inteligente de Entregas"**, applying the **SCRUM methodology**.  
The main goal is to plan and organize the development of an intelligent information system for logistics and urban delivery management.

The system is designed to support:
- Delivery planning and monitoring
- Customer, order, and courier management
- Resource and route optimization
- Customer experience improvement
- Decision support through indicators and data analysis
